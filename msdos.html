MS-DOS: The Early Microsoft Operating System

MS-DOS (which stands for Microsoft Disk Operating System) is an operating system which first came out in 1981 and was designed to run on PC compatible computers. It was primarily used as the main operating system on PCs until 1995, when Windows 95 overtook it as the most popular OS of choice.

MS-DOS was not originally created by Microsoft. Instead, IBM approached Microsoft to provide an operating system for its new personal computer. Microsoft acquired an existing OS called 86-DOS from Seattle Computer Products, modified it, and rebranded it as MS-DOS for IBM's use.

MS-DOS marked a turning point for Microsoft and the entire computer industry. When IBM needed an operating system for its new PC, Microsoft provided what became MS-DOS, but the crucial detail was that Microsoft kept the licensing rights.

This meant the company could sell the system to other manufacturers as well, allowing MS-DOS to spread rapidly across the growing PC market. That strategic decision helped establish Microsoft as a dominant force in personal computing for decades.
